Maya

This page will go over the buttons in the Maya shelf and what they do.

Assignments

This is where the students should start their assignments. Maya will ask where to save the file and then ask the user to choose the assignment. Maya will add the rig needed for the course, a camera to rendering and name the file correctly for the assignment and for the corresponding movies.

Creating an Assignment walk-through

Rigs

This references in any of the rigs used in the assingments. This will ADD a character to a scene and not start a new one.

Click on the image OR the checkbox to select a character. Multiple characters may be selected.

  • Reference Selected Characters - will reference ONLY the selected characters into a scene.
  • Reference Selected Characters and Camera - will reference the selected characters AND a camera into a scene.
Face Cam

Adds a camera to the head of a character. The camera will move with the head, creating a camera to always see the facial controls.

There are multiple ways to add a face cam. They all are similar.

Preferred Way

  1. Select ONLY the head controller of the character
  2. Click on the Face Cam button
  3. Brings up this dialog box
  4. Before clicking on the button, make sure to translate the camera in front of the character.
  5. Change one of your panels (preferred TOP LEFT PANEL) to FaceCam
  6. Translate the camera (using the W key to translate NOT ALT+Left mouse button!) to get the face in the center of the frame. When the camera is in the right place, go to the next step.
  7. Select the Head Controller
  8. Click on Attach Camera to Head
Scene

Brings up Maya's Reference editor. You can see the references in the scene. Clicking on the checkbox will only hide them from the scene.

Incremental Save

Will save your maya scene, not movie, up a version. If you were on V3, it will save the file as V4.

Studio Library

The library that holds poses and some animation to be reused.

Picker

Picker to select controls on each rig.

Designer

To create Pickers (Rarely used in animation classes)

Playblast

The ONLY way to create any movies for the assignments.

Creating a Movie to Submit OR for yourself
