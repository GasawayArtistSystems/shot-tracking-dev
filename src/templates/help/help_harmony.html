Storyboard Pro Pipeline

TIP: This next step is for ANY FILM SHOT in SBPro. Whether it's a new shot or if you are working on a current shot.

Creating Thumbnails

Click on the UC_SB_Creator

Select the film and scene you're working on.

📌 NOTE: If your scene isn’t in the dropdown, notify your UPM or instructor.

Click Open or Create Thumbnail or Storyboard.

New Thumbnails

A new file is created automatically. If a version already exists, a popup will show:

  • Open opens the current file (ok if it’s your own).
  • Create duplicates it and bumps the version (best practice).

Storyboard Pro will open and set up all folders and naming correctly.

Adding Audio

  1. Click the audio track
  2. Go to File → Import → Sound Clips
  3. Navigate to: D:/Films/FILM NAME/Audio/For Scenes
  4. Select the WAV for your scene
  5. Click Current Audio Track checkbox and OK

Naming the Project

Press ALT + RETURN or go to Storyboard → Properties. Change the title to the name of your film and click OK.

Naming Shots (Panels)

Name the first panel 010. Increment by 10 for each shot (020, 030, etc).

Creating Panels and Shots

New Panel:

Make sure the current panel has a red outline, then press P.

New Shot:

Click outside the current panel, then press P.

Change Layer Name

Rename the A layer to THUMB.

Exporting Movie – Thumbnails

Go to File → Export → Movie. Save to: D:/Films/FILM NAME/Thumbnails

📌 File Pattern: %t_010_THUMB_v1

Format: H.264/MPEG-4 (.mov)

Export Range: All

Burn-in Options:

  • Print Scene names and Panel numbers

Camera Grids:

  • Title Safe Area

Storyboards

Repeat steps using the same UC_SB_Creator tool. If thumbnails are approved, the tool will guide you into storyboard mode.

  • Open: Opens thumbnails (not typical for new SB)
  • Create SB: Duplicates thumbnails and prepares SB version

Exporting Movie – Storyboards

Same export steps as thumbnails, but...

📌 File Pattern: %t_010_SB_v1

Save in: D:/Films/FILM NAME/Thumbnails (yes, still Thumbnails folder!)

All export settings and options remain the same as above.
