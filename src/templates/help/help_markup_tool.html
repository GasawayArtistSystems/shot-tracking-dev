Markup Tool

This tool allows the user to select any movie or drawing to draw on the frame or drawing itself. It's also used to grade assignments and work with approvals.

Sections

The Markup Tool is broken up into 5 sections:

  1. SideBar
  2. Grader/Approver
  3. Player
  4. Timeline
  5. Tools

SideBar

The far left of the Markup Tool is the SideBar. This is where the user can click on which video or drawing file to add to the Player. These files are organized by class and assignment.

📌 NOTE: This is collapsible to give more room to review. The hotkey is ALT+b.

As soon as a shot or assignment has been submitted on the app or submitted through Maya/Harmony/SB Pro, it will appear here.

  1. Search Files
    1. The user can type in a student's name or the name of a shot here. It will list all the files with that name in it.
  2. Assignments to Review
    1. All assignments that need to be reviewed will be listed here. Simply click on the file and it will load into the player.
  3. Shots to Review
    1. All shots that need to be reviewed will be listed here. Simply click on the file and it will load into the player.
  4. Classes
    1. Under Classes will be all the available classes for the semester and then all movie/drawing files that have been reviewed.
    2. To watch a movie/drawing, click to the class where the file is and just click on the file. It will load into the player.
  5. Films
    1. Under Films will be all the available Scenes and Shots that have been reviewed. It's broken down by scenes and then shots.
    2. To watch a movie/drawing, click to the scene and then shot where the file is and just click on the file. It will load into the player.

Grader/Approver

This section has the name of the file and the status. For assignments, it's where the instructor can change the grade. For a shot, it's where the director can approve or retake.

After doing any markups, if needed, change the status and hit SAVE.

📌 NOTE: If no marks are used for the assignment or shot, a warning will pop up, asking if you still want to save.

Player

This is where the video will play or the image will be seen. A user can draw on top of this.

Timeline

A user can play, scrub, advance a frame or a keyframe at a time. Audio can be muted and the video can loop.

  1. Scrubber

    A user can scrub the scrubber throughout the entire animation.

  2. Tick Marks

    When a mark is drawn on the player, a yellow tick mark will appear.

    ALT+, will move to the next fraame

    ALT+. will move to the previous frame

    , will move to the previous keyframe

    . will move to the next keyframe

  3. Play

    Will play and pause the animation (Use the spacebar)

  4. Loop

    Will toggle the loop.

  5. Speaker

    Will mute/unmute the audio

  6. Volume slider

    Will adjust the volume of the audio

  7. Frame Counter

    Will show the frame number

Review Session

The tools for the review sessions, separate from reviewing individual assignments.

  1. Start Review Session

    If looking at an entire scene, it's best to start here. This will open a new modal.

    Select the Film, Scene number, and Step.

    Click start. This will open the Review Session on the top middle of the markup tool.

  2. Title

    Says what Film and Scene the session is for: IE - testme (film name) Scene 020.

  3. Prev Shot

    Will go to the previous shot unless you are on the first shot of the session.

  4. Next Shot

    Will go to the next shot in the session until the last shot.

  5. Make Combined Clip

    This will create a movie file with all the shots back to back.

    Once it's done, the user can click on the Open Review Clips button on the far left.

    That will open a new window:

    A user can play, scrub, or delete the file.

Tools

All the tools for the markup tool.

  1. Drawing tool (B)

    This will toggle between Start Drawing and Stop Drawing. Hit the B key to toggle.

  2. Brush Color

    There are four set colors to use. Click on the color to change the pen brush color.

    A user can also make a custom color by clicking on the rectangle.

  3. Brush Size

    Use the slider to change the brush size.

  4. Clear

    Will clear the markups for the frame you are on.

  5. Clear All

    Will clear all the markups for the entire timeline.

  6. Ghosting(Onion Skin)

    Can have ghosting for frames before and after.

    Toggle ghosting on and off with the checkbox.

  7. Frame Note

    Allows instructor/coordinator to add a note on selected frame.

Frame Note

Go to the frame to add a note. Type in the note and hit Save Note

This will add a GREEN LINE on your timeline.

OBS

If you use the OBS tool to screen record, these tools will help you turn on/off the record button in obs.

Note: If using OBS, it's necessary to run the StartOBSBridge.bat. This file will read the information from the file you are viewing so it names the file correctly. Once started, you can start/stop obs as below. After all reviews are done, Click on the Review Manager button (both files must be installed separately)

  1. Start OBS Recording

    Will start the recording on obs.

  2. Stop OBS Recording

    Will stop the recording on obs.

  3. Export Grades

    Will export a csv to be imported into Canvas.

Approval Sessions

For an approval session, the application will have everything ready to be viewed/approved.

There are two different ways to handle FILM approval sessions. You can do them one at a time, as described below. This behavior is the same for assignments. OR you can use the Start Review Session button.

  • Click either Assignments to Review or Shots to Review
  • Click on the assignment/shot to review - it will load into the player.
  • Make any markups if needed.
  • Change grade/status.
  • Click Save.

This will change the status in the overall application and move the file into the Classes folder or the Films folder.

    💡NOTE: The software will change the name of the file to add a "_R" near the end of the file. This is so the application knows it's been reviewed

    Viewing Movies

    To watch already reviewed movies, click on the name of the assignment or film and find the correct file. When you click on the file, it will load it into the player.

    If there are notes, it will show those notes as well.

    Deleting Movies

    It can be overwhelming with a LOT of movie files filling up the sidebar. A user can delete any file by right clicking on it.

    WARNING: ONLY let your UPM or coordinator delete a file as this is UNDOABLE!

    To delete a file from the sidebar AND from the folder,

    • Right Click on the file you want to delete.

    • Click on Yes, Delete it! - to delete it.

    • The file is deleted from the sidebar and the folder.

    Installing OBS Tools

    In order to use the OBS tools in the Markup tool, the user must download and install the OBS app and the UC_OBS_TOOLS from the server.

    These files will allow the user to use the start/stop obs tools by naming the files correctly - Assignment_Student name_DATE.wmv

    The instructor can then copy all the reviewed files for the day by using the Review Manager.

    1. Copy Local to Server

      Copies the entire Reviewed folder, as chosen by the user, to the server. This allows the dashboard to let the student know if a file is to be reviewed.

    2. Delete Local Folder

      Will delete the folder on your local machine

    3. Delete Server Folder

      Will delete the folder on the Server. Be careful not to delete any other instructors folder. If unsure, don't delete.
