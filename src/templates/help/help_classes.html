Classes

This page shows you how to manage classes, including viewing, adding, editing, and deleting classes. Classes hold all the assignments, grading tasks, and student records.

Classes are comprised of Assignments. These assignments are built based on Individual Assignmets that are attached to each student. Each class has assignments, students, and individual assignments.

View Classes

Clicking on the CLASSES link at the top of the page takes you to the main classes page, where you can view all existing classes, including their names, codes, semesters, and instructors.

Adding a Class

When you add a class, you are also adding the folders for that class on the server. This helps with the other buttons in (Maya, Harmony, and Storyboard Pro).

Clicking the Add Class button will open this window:

Follow these steps to create a class:

  1. Select the Semester
  2. Change the Course Code
  3. Enter the Class Number
  4. Enter the Class Name
    5. 📌 NOTE: The Class Preview is auto-generated and not editable. It shows the full name of the class for tracking purposes.
  5. Change Instructor
    6. 📌 NOTE: Instructors can be added by the ADMIN under the View Users section in the Admin dropdown.

Editing a Class

To edit a class, click the Edit button next to the class. This opens a modal where you can update the class name, semester, instructor, and other details.

📌 NOTE: If you change the name of the class or the semester in which the class is, a warning will ask if you want to add the folders and delete the old folders.

Click Yes to delete the old folders and create new ones or No to simply add new folders.

Deleting a Class

To delete a class, click the Delete button next to the class. Note that this action will also delete all associated assignments and cannot be undone.

⚠️ WARNING: Deleting a class will permanently remove all related assignments and student data.

Adding Students to a Class

To add students to a class, click the Students button on the main class page. This opens the student selection screen, where you can assign students to the current class.

Removing Students from a Class

To remove students from a class, select the student from the right column and click the Remove from Class button.

Adding Assignments to a Class

On the main Classes page, click on the desired class, then click the Add New Assignment button.

  1. Enter Assignment Name, Start Date, and Due Date.
  2. Select the workflow for the assignment.
  3. Choose Progress Tracking Steps to monitor assignment progress.
  4. Select one or more tracking options.
  5. Choose Assign to All Students.
    6. 📌 NOTE: Use "Assign to All Students" when first creating assignments. Use "Assign to Selected Students" when adding late joiners.

This action creates both the main assignment and student-specific assignments.

Editing Assignment

On the main Assignments page, click on the Edit button next to the desired Assignment

This will allow you to edit the Assignment name, Start Date, and Completion Date.

Deleting Assignment

On the main Assignments page, click on the Delete button next to the desired Assignment

A warning will pop up allowing you to delete the assignment or to cancel.

SUBMITTING ASSIGNMENTS

Trying to figure out what rigs are used for assignments is in the past. For Maya, the rigs are pre-loaded as long as the student selects the correct class and assignment. Students don't have to worry about naming convention or adding their name to the file. The code in Maya or ToonBoom will do it for them.

Software:

  • Maya
  • Harmony
  • Storyboard Pro

Maya Pipeline

Students working in Maya should use the GAA or DMC shelf tools to set up, save, and submit assignments using the following steps.

📌 NOTE: Before starting an assignment, in order for this submission to work, students MUST use the following steps.
  1. Click the Assignments button in the GAA shelf and save the Maya file in a folder you will remember.
    📌 REMEMBER: It's best to create a folder in your Documents folder OR on Onedrive. THIS is where your file will be saved. When opening an assignment, this is where it will be.
    NOTE: You don't have to type anything in the Directory: box. Once you go to your folder, you can simply click SAVE.
  2. Select your Class and Assignment from the modal window that appears.
  3. Click Create Assignment File. This loads the rigged scene and auto-generates the correct file name.
  4. A render camera is added to your scene.
    📌 REMEMBER: Do not delete this camera — it's used for rendering!
  5. Use File → Save to save your scene.
  6. You can version up with the Increment button.
  7. Create a Playblast using the GAA Playblast button.
  8. Click Movie For You to generate a local .avi file (not for submission).
  9. To submit: click Submit Assignment. The movie file is placed automatically in the correct folder.
  10. If audio is detected, check the audio box to include it.
📌 REMEMBER: Track assignment status anytime from your Dashboard. You can update your status after grading.

Revise and Resubmit

Making changes and then resubmitting is the exact same process as creating an assignment and creating a movie. If your assignment is less than a 5, the software will make the status of your assignment a RETAKE.

  • IF your status is RETAKE, you have the option to make changes and get a better grade or be happy with the grade and change the status to Approved.
  • IF you choose to make changes, simply follow the same directions as Creating a Shot.

Config Editor

In order to make Maya, Harmony, and Storyboard Pro compatible with this application, we must create a config editor. This editor will show all the classes and assignments, along with the Rigs that are used in each assignment. Follow the intuitive instructions to creating a config file for each software.

📌 NOTE: It's best to do the config editor after creating the film and scenes/shots/assets as this editor gets all the information from those scenes/shots/assets.
⚠️ WARNING: This should ONLY be done by an instructor with all other instructors aware, at the beginning of each semester. All tools are based on this editor. It will not be able to be used after the semester starts.
  1. To speed the process, Load an Old Config. This will allow you to locate an old config file so you can edit that instead of starting from scratch.
  2. Select Semester from dropdown.
    3. 📌 NOTE: If the semester is different than is in the initial dropdown, you will get a warning, saying the Switching will reset everything. When first starting, this is ok. Select Yes, Discard changes.
  3. Click the + Add Class button on the main modal and select a class you want to add.
  4. Click + Add Assignment button under the class you want the assignment to be.
  5. Select an assignment from the dropdown or add your own.
  6. The editor is populated with the Assignment. You can now add if you want a camera by clicking the checkbox and add rigs by clicking on the Select Rigs button.
  7. In the Select Rigs modal, you can click on multiple checkboxes, or do a search for the rig. Rolling over the rig name will give you the full name.
  8. Continue to add classes/assignments/rigs until finished.
  9. You can move assignments in the order you want by clicking and dragging on the assignment. Classes can be ordered the same way.
  10. You can delete assignments or classes by clicking the red trashcan icon on the far right.
  11. Once complete, you can Save a Draft to continue working on later or Save JSON to save the config file
    12. NOTE Saving the JSON file will put the config file in the correct place in Maya. Nothing else needs to be done.
