Storyboard Pro Pipeline

The process will go from thumbnails, exporting thumbnails, approvals, storyboards, exporting storyboards, approvals, exporting to Harmony.

The toonboom tools for both SBPro and Harmony will be on your desktop.

TIP: This next step is for ANY FILM SHOT in SBPro. Whether it's a new shot or if you are working on a current shot.

THUMBNAILS

Creating Thumbnails

  1. Click on the UC_SB_Creator

    This will open a modal:

  2. Select the film from the drop-down menu
  3. Select the Scene that you are working on.
📌 NOTE: If your scene isn’t in the dropdown, notify your UPM or instructor.
  1. Click on Open or Create Thumbnail or Storyboard

Different ways that the tool can create or open files.

This tool is smart enough to know what files are already created and approved. All you must do is know what shot and what step you are working on: thumbnails or storyboards.

New Thumbnails

Follow the above steps. A new file will be created without a prompt after you click on Open or Create Thumbnail of Storyboard.

Revision of Thumbnails

If you are working on a revision of thumbnails that you or someone else worked on, go through the exact same steps. The difference is when you click on the Open or Create Thumbnails or Storyboard.

  • Open opens the current file (ok if it’s your own).
  • Create duplicates it and bumps the version (best practice).

Storyboard Pro will open and set up all folders and naming correctly.

📌 NOTE: There are a couple of housecleaning steps to do before you start drawing. It's imperative that you do this!

Adding Audio

Most of the time, there will be audio for your scene. It's only to be used as a guide and you can go off it, if needed. It's created by the editor and director in the form of the radio play.

  1. Click the audio track
  2. Go to File → Import → Sound Clips
  3. Navigate to: D:/Films/FILM NAME/Audio/For Scenes
  4. Select the WAV for your scene
  5. Click Current Audio Track checkbox and OK
📌 NOTE: You may split/edit/move the audio track wherever you like. It will be preserved in the mov file you create.

NAME OF FILM

For our tools to work, you MUST name your project correctly.

Naming the Project

  1. Change the Project Title by clicking ALT + RETURN or go to Storyboard → Properties
  2. This brings up the Properties window
  3. Change the Project Title to the name of the film you are working on
  4. Click OK

NAME OF SHOT

📌 NOTE: Remember, although Storyboard Pro names the shots, SCENES, we will treat them like SHOTS

Naming Shots (Panels)

📌 NOTE: You MAY NOT NEED to change the name of the Shot. The preferences should have been added. IF NOT, follow the directions that follow:
  1. Change the name of the shot by changing the name of the scene in the PANEL tab on the far right.
  2. Make sure to start your first panel named 010 ← don't forget the 0 in front. Each shot must have THREE digits.
Each new shot will be named 020, an increment of ten with the 0 in front. 010, 020, 030, and so on until 100, 110, 120, and so on.

Creating a new PANEL for Shots

To create a new panel, make sure you have your timeline scrubber IN the current panel. It will have a red outline around it.

  1. Make sure the current panel has a red outline around it
  2. Press P on your keyboard

All the panels for each shot will look grouped in the timeline.

Creating a new SHOT

To create a new shot, make sure you have your timeline scrubber OUTSIDE of the current panel. No panel will have a red outline around it.

  1. Click the scrubber outside of any panel.
  2. Press P on your keyboard
📌 NOTE: Be sure to change the name of the shot to the next number of ten (IE - 010, 020, 030, 040...100, 110, 120...)

Changing the name of the LAYER

When you create a new shot, or a panel for a new shot, make sure to change the top layer from A to THUMB.

Rename the A layer to THUMB.

📌 NOTE: This will help when you are doing final storyboards.

EXPORTING

Instead of exporting images or pdfs, it’s better for our markup system to use movie files. This also allows boarders to work on timing in their shots.

Exporting the Thumbnails

  1. Go to File → Export → Movie
  2. Change the Destination path to your Thumbnails folder:
    C:/Films/FILM NAME
  3. File Pattern: %t_010_THUMB_v1
    4. 📌 NOTE: Make sure that the 010 is YOUR scene. If you are working on scene 050, make sure the name is: %t_050_THUMB_v1
  4. Set the Movie Format to H.264/MPEG-4 (.mov)
  5. Set the Export Range to All
  6. Under Burn-in Options, check the following:
    • Print Scene names and Panel numbers
  7. Under Camera Grids, check the following:
    • Title Safe Area
