Films

This page will show how the films page works. It shows how to view the films, add films, edit films, and delete films. The films hold all the pre-production, scenes, and assets. It also holds all the statuses to track the progress of the film through all stages.

Films are made up of Scenes and Assets. Scenes are made up of shots. Assets can be: BGs, Sets, Rigs/Characters, 2d props, or 3d props.

View Films

Clicking on the FILMS link at the top of the page takes you to the main films page, where you can view all existing films, including their names, stage of production, main crew, shots and asset completion.

Adding a Film

Clicking the Add Film button will open this window:

Follow these steps to add a film:

  1. Select the Semester
  2. Enter the film name
  3. Enter the description, if needed.
  4. Select Overall Film Flow
    5. 📌 NOTE: This overall flow is the workflow of the film. Most use 2d Movie Flow or 3d Movie Flow.
  5. Select Director Name
  6. Select UPM
    7. 📌 NOTE: The names in the dropdown must be added by the Administrator.
  7. Click Add Film

Editing a Film

To edit a film, click the Edit button next to the film. This opens a modal where you can update the semester, film name, description, workflow, director, and UPM.

Deleting a Film

To delete a Film, click the Delete button next to the Film. Note that this action will also delete all associated assignments and cannot be undone.

⚠️ WARNING: Deleting a film will permanently remove all related shots and asset data.

Adding Crew

Clicking the Crew button will open this window:

  1. Select names for the coordinator positions.
  2. Click Checkboxes for artists working on the film.
    3. 📌 NOTE: If your coordinators or artists names aren't on the list, contact your instructor to add them.
  3. Click Save Crew.

View Scenes

Clicking on the SCENES to the right of the film in which you want to view its scenes, will take you to the scenes page.

This page shows each scene and all the statuses for each step of the production. If you hover overtop one of the pie charts, you can see the exact number of shots at each stage.

Scene Header

The Scenes header allows you to switch between films, scenes, and assets.

  1. Films Dropdown - let's you quickly go from one film to another
  2. Films button - goes back to the main film page
  3. Scenes button - goes to the scenes page for the selected film
  4. Assets button - goes to the assets page for the selected film

Show Filter/Hide Filter - allows you to access the Filters dropdown.

  1. Show Filter/Hide Filter - a toggle between opening the filter and closing it
  2. All Steps - Shows every available step in the process
  3. Artist Steps Only - Shows the main steps for artists. Doesn't include the FB steps
  4. Clickable checkboxes - A user can click on/off any step

Adding a Scene

Clicking on the + Add Scene on the top right of the Scenes page, will open an Add Scene modal.

Follow these steps to add a scene:

  1. Add Scene Number - this MUST BE three digits
  2. Change Start Date - This will be filled in by the dates in the timeline. You can still change them
  3. Change Due Date - This will be filled in by the dates in the timeline. You can still change them
  4. Enter a description
  5. Include Thumbnail Step - this is clicked ON by default. If you don't want thumbnails, click it off.

Adding Multiple Scenes

Clicking on the + Add Mutliple Scenes on the top left of the Add Scene modal.

Follow these steps to add multiple scenes:

  1. Start Prefix - this MUST BE three digits
  2. Enter how many scenes to add.
  3. Enter a description (optional)
  4. Change Start Date - This will be filled in by the dates in the timeline. You can still change them.
  5. Change Due Date - This will be filled in by the dates in the timeline. You can still change them.
  6. Include Thumbnail Step - this is clicked ON by default. If you don't want thumbnails, click it off.

Editing a Scene

To edit a scene, click the Edit button next to the scene. This opens a modal where you can update the scene number, start date, due date, and description.

Deleting a Scene

Clicking on the Delete Scene to the right of the scene itself, will warn if you want to delete the scene or not.

⚠️ WARNING: Deleting a scene will permanently remove all related shots.

View Shots

Clicking on the SHOT NUMBER to the left of the scenes table, will take you to the shots page of that scene.

This page shows each shot and all the statuses for each step of the production. This is where a user can change the status, change the assignee, and the due date.

Shots Header

The Shots header allows you to switch between films, scenes, and assets.

  1. Scenes Dropdown - let's you quickly go from one scene to another
  2. Films button - goes back to the main film page
  3. Scenes button - goes to the scenes page for the selected film
  4. Assets button - goes to the assets page for the selected film

Show Filter/Hide Filter - allows you to access the Filters dropdown.

  1. Show Filter/Hide Filter - a toggle between opening the filter and closing it
  2. All Steps - Shows every available step in the process
  3. Artist Steps Only - Shows the main steps for artists. Doesn't include the FB steps
  4. Clickable checkboxes - A user can click on/off any step
  5. Dropdowns - A user narrow down what they want to see by the dropdowns

Adding a Shot

Clicking on the + Add Shots on the top left of the shots page, will open an Add Shot modal.

Follow these steps to add a Shot:

  1. Add Shot Number - this MUST BE three digits
  2. Enter a description
  3. Add Start Date - This will be filled in by the dates in the timeline. You can still change them.
  4. Add Due Date (optional)
  5. Assign to (optional) -- this is for the ENTIRE shot, not just a step. This is RARELY used.
📌 NOTE: Use "+ Add Multiple Shots" to add more than one shot at a time. This is best to use when first creating shots.

Adding Multiple Shots

Clicking on the + Add Multiple Shots inside the Add Shot modal will bring up another modal.

Follow these steps to add Multiple Shots:

  1. Add Prefix - this is what the first shot number will be - 050, 130, 520. It must be three digits with the last number being a zero.
  2. How many shots? - the total number of shots you are adding.
  3. Description (optional) RARELY used.
  4. Add or Cancel to add the shots or cancel out of the modal.

Editing a Shot

To edit a Shot, click the Edit button to the far right of the shot. This opens a modal where you can update the shot number, start date, due date, and description.

Deleting a Shot

Clicking on the Delete Shot to the far right of the shot, will warn if you want to delete the shot or not.

⚠️ WARNING: Deleting a shot will permanently remove the shot from the database.

Bulk Editing Multiple Shots

Editing multiple shots is a powerful way to edit a lot of shots in one place. This can change statuses, assignees, and/or due dates.

Follow these steps to Edit Multiple Shots:

  1. Click on the checkboxes of the shots you want to edit.
  2. Click on top far right of the Shots page.
  3. That opens a new modal:
  4. Change the status, assignee, and/or due dates for whatever steps you want.
  5. Click Save Changes to make the changes, or Cancel to get out of the modal without changes.

Assigning single shots

A UPM or coordinator can assign a single shot.

  1. Locate the shot.
  2. Click on the Assignee tab (--Unassigned--).
  3. Select the artist's name

View Assets

Clicking on the ASSETS to the right of the film in which you want to view its assets, will take you to the assets page.

This page shows each type of asset and all the statuses for each step of the production. If you hover overtop one of the pie charts, you can see the exact number of assets at each stage.

Clicking on one of the categories to the left will take you to all the assets in that category.

Adding an asset

Clicking on the Add New Asset on the top right of the Assets page, will open an Add Asset modal.

Follow these steps to add an asset:

  1. Enter Asset Name
  2. Select a Category from the dropdown
  3. Change Due Date - This will be filled in by the dates in the timeline. You can still change them.

Editing an Asset

To edit an asset, click the Edit button next to the asset in the asset category page. This opens a modal where you can update the asset name, category, and due date.

Deleting an Asset

A user can delete one or more assets at a time.

  1. To delete an asset, you must click on the checkbox to the far left of the assets. You can click multiple assets for this.
  2. Click on Delete Selected button
  3. Click Yes if you want to delete it or cancel to not delete it.

To delete an asset, you must click on the checkbox to the far left of the assets. You can click multiple assets for this.

⚠️ WARNING: Deleting an asset(s) will permanently remove the asset(s) from the database.

View Assets Category

Clicking on the ASSET CATEGORY to the left of the table on the Assets page, will take you to that category page.

This page shows each step of each asset in the category. A user can edit each status or select multiple assets to edit them as a group.

There is also an icon next to the asset name that shows if the asset is available as an asset to be used in production.

The X means it hasn't been created or saved and the folder icon means it is available to be used in production.

Assets Location on Server

The actual asset files are located on the local server. The directory is C:/Films/NAME OF FILM/Assets/TYPE OF ASSET/name of the file

Note: The directories are created when the Film is created. The asset files are created and edited through either Maya or Harmony.

SUBMITTTING SHOTS/ASSETS

This takes the guesswork out of figuring out where your shots are saved. This process is super simple but make sure to follow the steps carefully.

Software:

  • Maya
  • Harmony
  • Storyboard Pro

Shot Walk-through

The coordinator or UPM will assign a shot with a due date.

The user will then go to their dashboard to see what is all assigned to them and when they are due. When the user starts working on the shot, they should change the status to IN PROGRESS.

Now the user is ready to work on the shot or asset, in Maya, Storyboard Pro, or Harmony.

Maya Pipeline

TIP: This next step is for ANY FILM SHOT in maya. Whether it's a new shot or if you are working on a current shot.

Shots

Creating a shot for the film.

  1. Open Maya
  2. Click on the Film Shot button in the GAA shelf
    1. This will open a window to simply let you select the shot you are working on.
    2. Select the Film from the film dropdown.
    3. Select the Scene from the dropdown and then the shot from the dropdown.
    4. Select the STEP in the process you are working.
    5. Click on Create Shot File
      6. 📌REMEMBER: The name of the file will show up just above this button. Make sure it's correct!
    6. This will load the file, rigs, and any props that are used in the shot. It will also save the scene to the correct location.
  3. IF a file exists, the software will give a warning.
    1. Open Existing - If you want to work on the current version, click on this button. It will load the current version.
    2. Open Older - This will allow you to open an older version. It will take you to where the files are located. Pick which version you want to work on.
      3. NOTE: USUALLY, you will just click on OPEN EXISTING. Be very careful with this step. It's best to ask your coordinator.
    3. New Version - This starts a brand new scene with nothing in it. Only use this if directed by your coordinator.
    NOTE: If you are working on the next Step - IE Animation after layout is completed, a different warning will popup.
  4. If you want to work on the current step, click on Open Previous. IF moving to the next step, Click on New (step) shot. This will open the last step and rename it correctly.

Creating a PLAYBLAST, or MOVIE file

  1. Click on the GAA Playblast button
    1. This will open the playblast window. It's SUPER EASY. It will name your file and put in the place where you told the Assignment button to save your maya file. It will save your movie files in the correct places.
    2. To generate a movie FOR YOU - NOT to submit!
      1. Simply Click on the MOVIE FOR YOU button
      2. This will save an .avi file in the same place where you save your scenes. No need to save it anywhere else. The code does it for you.
    3. To submit your shot for review:
      1. Click Override Step checkbox
      2. Select the step that you are submitting.
      3. Click SUBMIT SHOT
      4. That's it! The code will automatically put the movie file in the right place. It will also create an .avi file in the folder you selected when creating the assignment, so you can view when you like.
    4. If you have audio, the code will let you know it's in the file. Just click on the checkbox to include audio.
    📌REMEMBER: Need to see where your shot statuses are? Go to the DASHBOARD .
  2. The STATUS of your assignment is now switched to Submitted.
  3. You MAY check on the status OR change your status of your shot once it has been graded.

    4. Assets

    Creating an asset for the film.

    1. Open Maya
    2. Click on the Film Create Asset button in the GAA shelf
      1. This will open a window to simply let you select the asset you are working on.
      2. Select the Film from the film dropdown.
      3. Select the Asset Type from the dropdown.
      4. Select the Asset Name from the dropdown
        5. WARNING: If your asset isn't on the list, let your coordinator know!
      5. If this is the first time creating the asset, a blank maya scene will open. It will be named correctly and put in the right spot.
    3. Click on Create Asset File
    4. IF a file exists, the software will give a warning.
    5. How would you like to proceed?

        6. NEW VERSION

        This will create a COPY of the previous version but will have your username on it and it will bump up a version.

        NEW FILE

        This will create a brand new EMPTY FILE but will have your username on it and it will bump up a version.

        OPEN EXISTING

        This will open the current version but will replace your username and it will bump up a version.

    Config Editor

    In order to make Maya, Harmony, and Storyboard Pro compatible with this application, we must create a config editor. This editor will show all the films, scenes, shots, and assets that are to be used in the films for the semester. Follow the intuitive instructions to creating a config file for each software.

    📌 NOTE: It's best to do the config editor after creating the film and scenes/shots/assets as this editor gets all the information from those scenes/shots/assets.
    ⚠️ WARNING: This should ONLY be done by an instructor with all other instructors aware, at the beginning of each semester. All tools are based on this editor. It will not be able to be used after the semester starts.
    1. To speed the process, Load an Old Config. This will allow you to locate an old config file so you can edit that instead of starting from scratch.
    2. Select Semester from dropdown.
      3. 📌 NOTE: If the semester is different than is in the initial dropdown, you will get a warning, saying the Switching will reset everything. When first starting, this is ok. Select Yes, Discard changes.
    3. Click the + Add Film button on the main modal and select a film you want to add.
    4. Click + Add Scenes button under the Film.
    5. A window will open asking if you want to add All Remaining Scenes or No, let me choose. Adding will add all scenes, choosing will bring up another window.
    6. Click on the checkboxes next to the scenes you want to add.
    7. You can add shots by clicking the + Add Shots button.
    8. A window will open asking if you want to add All Remaining Shots or No, let me choose. Adding will add all shots, choosing will bring up another window.
    9. Click on the checkboxes next to the shots you want to add.
    10. Now you are ready to add assets. Each shot is divided so you can add assets to each shot.
    11. You can add assets by clicking the Select Assets for each Shot.
    12. Assets are available if you see a folder next to the name. If there is an x, that asset isn't production ready. You can still add it to the config file.
    13. Each asset will be shown in its own category. If you minimize the shot window, the assets will still be listed under the Select Assets button.
    14. You open/close or delete assets/shots/scenes by clicking the +- button or the x button to delete.
    15. Once complete, you can Save a Draft to continue working on later or Save JSON to save the config file
      16. NOTE Saving the JSON file will put the config file in the correct place in Maya. Nothing else needs to be done.

    Creating Folders

    For Maya and Toonboom to work with Shot Tracker, the user must create the folders needed for those applications. After you have created or edited the config file (see above), you can click on the Folders button on the main Film Page.

    A popup window will ask if you want to generate the folders. Take note of the date that the config file was created! You may need to update that config file first.

    Click on Yes, generate, to create the folders.

    Assigning Shots/Assets

    Assigning or casting is done through here. This page will help the UPM and coordinators figure out workload for each artist. This takes most of the guesswork out of assigning shots.

    Click on this Assign button to open the Assign window.

    The page is designed like the following.

    1. The header includes the Name of the film, Total Shots, how many are unassigned, and how many artists are on the project.
    2. To expand/collapse any folder, simply click on the expand/collapse button.
    3. Click the Artist Load button will show which artist has which shots. This is used ONLY as a visual reminder. You can't add/delete who works on the shot from here.
    4. Each heading will have the number of shots or assets in each department.
      5. 📌 NOTE: Make sure to add the difficulty of each scene and shot prior to assigning.
    5. Assign difficulty in Scenes-Thumbnails and in Animation
      6. 📌 NOTE: Assigning difficulty will help make shot assignment much easier.
    6. By clicking to expand each heading, you will see the shot or asset, assigned artist, due, status
    7. This is where you assign artists and the due date. The status will change in the Markup, not here.
    8. The best way to use this is to go back and forth between the Artist Load and each header.
