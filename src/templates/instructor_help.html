📘 Overview

Welcome to the Film and Class Tracker Guide. This page outlines how to create and run classes and assignments, track submissions, and explore and create workflows.

💡 Tip: You can link to any section using the anchor links in the TOC.

🛠️ How It Works

Students render playblasts and the system logs submissions if metadata is detected:

  • GAA_user: student login
  • GAA_class: class name
  • GAA_assignment: assignment ID
